# Api quản lý nhà trọ
## Sử dụng framework Laravel 8

### cách chạy api.
1. Sao Chép file .env.example sang .env
2. chạy: 
```bash
composer install
```
3. Run trong cmd: 
```bash
php artisan key:generate
```
4. chạy tạo cơ sở dữ liệu.
```bash
php artisan migrate
```
5. Chạy tạo dữ liệu mẫu.
```bash 
php artisan db:seed
```
6. chạy server localhost.
```bash 
php artisan serve
```
